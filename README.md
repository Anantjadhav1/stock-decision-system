# 📊 Stock Market Decision Support System

A real-time stock analysis terminal built with Flask, Data Structures, and ML.

## Features
- Real-time data via Yahoo Finance API
- Moving Average using Circular Queue
- Entry points via AVL Tree
- Fundamentals via Red-Black Tree
- ML price prediction (Linear Regression)
- Portfolio tracking
- Analysis history with persistence
- Bloomberg-style terminal UI

## Run Locally
pip install flask yfinance
python app.py
# Open http://localhost:5000
